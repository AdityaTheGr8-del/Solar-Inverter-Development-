# Project Vision

The ultimate goal of this project is to build a reliable, high-efficiency closed-loop solar inverter. To ensure a solid engineering foundation, the development is broken down into progressive phases.

### Phase 1: Open Loop (Current Focus)
* Generate stable SPWM (Sinusoidal Pulse Width Modulation) signals.
* Successfully drive the H-bridge utilizing IR2104 gate drivers.
* Step up the voltage using a transformer without feedback control.
* Document waveform outputs and switching characteristics.

### Phase 2: High Power Closed Loop (Future)
* Integrate voltage and current sensing for feedback.
* Implement safety algorithms (short circuit and overload protection).
* Optimize for higher load capacities.
